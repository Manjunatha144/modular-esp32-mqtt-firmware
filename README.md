\# ESP32 FreeRTOS MQTT Telemetry



\## Overview

ESP32 firmware project using ESP-IDF and MQTT.



\## Features

\- WiFi connection

\- MQTT publish every 5 seconds

\- Command subscription

\- Sequence number tracking

\- JSON telemetry payload
